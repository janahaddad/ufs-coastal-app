UFS Coastal Application Documentation
=====================================

The UFS Coastal Application contains a set of NUOPC-compliant data
and physical model components along with basic workflow to create
and configure different configurations of those models. It also
provides standardized interface to handle model dependencies and
porting application to different HPC platforms.
